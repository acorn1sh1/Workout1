# Workout Tracker App

A 5-day progressive overload workout tracker built with React Native + Expo.

---

## Setup (takes about 5 minutes)

### Step 1 — Install Node.js
Download from https://nodejs.org and install the LTS version.

### Step 2 — Install Expo CLI
Open Terminal and run:
```
npm install -g expo-cli
```

### Step 3 — Install app dependencies
Navigate to this folder in Terminal:
```
cd WorkoutTracker
npm install
```

### Step 4 — Start the app
```
npx expo start
```

### Step 5 — Open on your iPhone
1. Download **Expo Go** from the App Store (free)
2. Open Expo Go on your iPhone
3. Scan the QR code shown in your Terminal
4. The app opens instantly!

---

## Features
- 5-day split: Upper / Lower / Abs
- Log weight + reps for every set
- "Last week" hints so you know what to beat
- 4 progressive phases over 8 weeks
- Exercises automatically change at Week 5
- Form video links for every exercise
- Progress charts comparing week over week
- Data saved locally on your phone

## Program Structure
| Weeks | Phase | Focus |
|-------|-------|-------|
| 1-2 | Foundation | Learn movements, find starting weights |
| 3-4 | Load Up | Add 5-10 lbs to each lift |
| 5-6 | Volume | Harder exercises, more sets |
| 7-8 | Intensity | Push to near failure |

---

## Tips
- Rest 60-90 seconds between sets to stay within 30 minutes
- If you hit the top of the rep range easily → add weight next week
- Tap "▶ Form" on any exercise to watch a YouTube tutorial
